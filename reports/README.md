# Reports Alanı

Bu klasör resmi rapor taslakları ve çıktı varlıkları için ayrılmıştır.

## Alt Klasörler

* `PDR_OTR`: Ön tasarım raporu taslakları.
* `PCR_FTR`: Final tasarım raporu taslakları.
* `drafts`: Ara taslaklar.
* `final_assets`: Rapor için kullanılacak şekil, tablo, ekran görüntüsü.

## Not

Resmi `.docx` şablonları kök dizinde tutulur. Buradaki Markdown dosyaları rapor metni hazırlığı içindir.
