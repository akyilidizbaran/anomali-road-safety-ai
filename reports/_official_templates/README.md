# Official Templates

Resmi `.docx` ve `.pdf` dosyalarının bazıları kullanıcı referansını bozmamak için kök dizinde
tutulabilir. FTR teslim dokümanı ayrıca ASCII-safe adla bu klasöre kopyalanmıştır:

* `teknofest_2026_ftr_teslim_dokumani.pdf`

İleride root sadeleştirilecekse bu klasöre taşınabilir. Taşıma yapılırsa aşağıdaki dosyalar güncellenmelidir:

* `docs/README.md`
* `reports/README.md`
* `PROJECT_MEMORY.md`
* Rapor kaynak/şablon referansları.
