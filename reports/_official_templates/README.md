# Official Templates

Resmi `.docx` ve `.pdf` dosyaları şu anda kullanıcı referansını bozmamak için kök dizinde tutulmaktadır.

İleride root sadeleştirilecekse bu klasöre taşınabilir. Taşıma yapılırsa aşağıdaki dosyalar güncellenmelidir:

* `docs/README.md`
* `reports/README.md`
* `PROJECT_MEMORY.md`
* Rapor kaynak/şablon referansları.
