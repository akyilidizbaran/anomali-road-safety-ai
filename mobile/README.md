# Mobile Alanı

Bu klasör Android mobil uygulama planı ve ileride kodları için ayrılmıştır.

## Alt Klasörler

* `android`: Gerçek Android proje kodu için.
* `wireframes`: Ekran taslakları.
* `ui_requirements`: Ekran gereksinimleri ve kabul kriterleri.

## Ana Ekranlar

* Login.
* Camera.
* Evidence.
* System.
* Settings.

## UI Requirements

Ekran bazlı gereksinimler `mobile/ui_requirements/` altında tutulur.

* `camera_screen.md`
* `evidence_screen.md`
* `system_screen.md`
