# Mobile Alanı

Bu klasör Android mobil uygulama planı ve ileride kodları için ayrılmıştır.

## Alt Klasörler

* `android`: Gerçek Android proje kodu için.
* `wireframes`: Ekran taslakları.
* `ui_requirements`: Ekran gereksinimleri ve kabul kriterleri.
