# Settings Screen Requirements

## Required UI

* Backend endpoint configuration.
* Risk threshold.
* OCR threshold.
* Frame skipping option.
* QoD mock/real mode indicator.
* Number Verification mock/real mode indicator.
* Single-target / multi-target experiment toggle.

## Acceptance Criteria

* MVP default is single-target mode.
* Multi-target mode is marked as experimental.
* Settings do not expose secrets or API keys.
