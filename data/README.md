# Data Alanı

Bu klasör veri seti çalışmalarının yeridir.

## Alt Klasörler

* `raw`: Ham veri.
* `processed`: İşlenmiş veri.
* `annotations`: Etiket dosyaları.
* `splits`: Train/validation/test listeleri.
* `synthetic`: Sentetik plaka veya augmentation çıktıları.
* `README_assets`: Veri seti lisans ve açıklama dosyaları.

## Kural

Kişisel veri içeren ham dosyalar izin ve güvenlik politikası olmadan kalıcı saklanmamalıdır.

## Policy ve Template Dosyaları

* `DATA_POLICY.md`
* `DATASET_CARD_TEMPLATE.md`
* `README_assets/dataset_inventory_template.csv`

## Aktif Dataset Çalışmaları

* `README_assets/bdd100k_vehicle_detection_dataset_card.md`: BDD100K araç detection dataset card.
* `README_assets/bdd100k_vehicle_detection_mapping.yaml`: BDD100K sınıf ve condition mapping.
* `README_assets/dataset_inventory.csv`: aktif veri seti envanteri.
