# Araştırma 13 - Test Planı ve Metrik Sistemi

## Amaç

Final rapordaki “Çözümümüze neden güveniyoruz?” sorusunu somut metriklerle cevaplamak.

## Alt Başlıklar

* Görev bazlı metrikler.
* Sistem bazlı metrikler.
* Event-level metrikler.
* Araç tespiti: mAP, precision, recall, F1.
* Takip: IDF1, MOTA, ID switch.
* Plaka: plate accuracy, character accuracy.
* Hız: MAE, RMSE.
* Şerit: IoU, F1.
* Sürücü: F1, confusion matrix.
* Hava/görüş: macro F1.
* Sistem: FPS, latency, RAM, CPU/GPU.
* Demo senaryoları.
* Normal/kritik mod testi.
* QoD mock açık/kapalı testi.
* Evidence kayıt bütünlüğü.

## Çıktı

Final test matrisi ve metrik rapor şablonu.
