# Test Stratejisi

## Test Katmanları

1. Model bazlı test.
2. Pipeline bazlı test.
3. Event-level test.
4. Sistem performans testi.
5. Mobil UI akış testi.
6. Demo senaryo testi.

## Model Metrikleri

* Araç: mAP, precision, recall, F1.
* Tracking: IDF1, MOTA, ID switch.
* OCR: plate accuracy, character accuracy, edit distance.
* Hız MVP: relative motion consistency, motion anomaly event accuracy.
* Hız final scope: MAE, RMSE, km/s hata aralığı.
* Şerit: IoU, F1.
* Sahne: macro F1.
* Cabin: F1, confusion matrix.

## Sistem Metrikleri

* FPS.
* Latency.
* Model inference time.
* RAM/CPU/GPU.
* Evidence save time.
* WebSocket round trip.

## Event-Level Metrikler

* Doğru event üretim oranı.
* Yanlış alarm oranı.
* Kritik olay kaçırma oranı.
* Evidence completeness.
* QoD aday karar doğruluğu.
