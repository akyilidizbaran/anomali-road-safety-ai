# Rapor Format Kılavuzu

## Resmi Format

* Kapak ve İçindekiler ayrı sayfa.
* 3-10 sayfa sınırı.
* Arial 12 punto.
* Başlık Arial Black 14 punto.
* Satır aralığı 1.15.
* İki tarafa yaslı.
* Kenar boşlukları: üst 2.8 cm, alt/sağ/sol 2.5 cm.

## Rapor İçin Önerilen Şekiller

* Sistem blok diyagramı.
* Normal/kritik mod akış diyagramı.
* AI pipeline diyagramı.
* Event JSON alan özeti.
* Evidence kartı örneği.
* Test metrik tablosu.

## Dil

* Kesin olmayan iddialar “hedeflenmektedir” diye yazılmalı.
* Mock servisler gizlenmemeli.
* Hukuki karar dili kullanılmamalı.
* Aynı cümleler tekrar edilmemeli.
