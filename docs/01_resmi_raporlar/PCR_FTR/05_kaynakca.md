# 5. Kaynakça

## Resmi Beklenti

Kullanılan akademik ve teknik kaynaklar detaylı listelenmelidir. GitHub repoları, makaleler, teknik dokümantasyonlar ve veri setleri kaynakçada yer almalıdır.

## Kaynak Kategorileri

* YOLO ve object detection kaynakları.
* ByteTrack, BoT-SORT ve tracking kaynakları.
* OCR ve plaka tanıma kaynakları.
* Lane detection kaynakları.
* Sahne/hava/görüş veri setleri.
* Android CameraX ve Jetpack Compose.
* FastAPI, WebSocket, ONNX Runtime.
* 5G Number Verification ve QoD API dokümantasyonları.
* KVKK ve etik veri işleme kaynakları.
