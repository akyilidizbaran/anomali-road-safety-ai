# Testing Alanı

Bu klasör test senaryoları ve metrik raporları için ayrılmıştır.

## Alt Klasörler

* `metrics`: Metrik tanımları.
* `reports`: Test raporları.
* `scenarios`: Demo/test senaryoları.

## Ana Testler

* Model bazlı doğruluk.
* Pipeline FPS/latency.
* Event-level doğruluk.
* Evidence bütünlüğü.
* Mobil akış testi.

## Template Dosyaları

* `templates/vehicle_detection_metrics.csv`
* `templates/ocr_metrics.csv`
* `templates/system_latency_fps.csv`
* `templates/event_evidence_tests.csv`
