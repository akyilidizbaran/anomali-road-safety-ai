# Benchmarks

Küçük benchmark CSV/JSON/Markdown sonuçları takip edilebilir. Büyük log ve artifact dosyaları `artifacts/` altında lokal kalmalıdır.

Aktif benchmark alanları:

* `vehicle_detection_comparison.csv`: vehicle detector adaylarının karşılaştırması.
* `tracking/tracking_baseline_comparison.csv`: ByteTrack, BoT-SORT ve opsiyonel tracker adaylarının karşılaştırması.
* `condition_experts/`: condition profile ve condition expert benchmark şablonları.
