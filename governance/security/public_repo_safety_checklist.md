# Repository Safety Checklist

Commit öncesi kontrol:

* [ ] `.env` veya secret dosyası yok.
* [ ] API key yok.
* [ ] Android keystore veya signing dosyası yok.
* [ ] Raw video yok.
* [ ] Evidence image/crop yok.
* [ ] Plaka/yüz/cabin görseli yok.
* [ ] Model checkpoint/export yok.
* [ ] Dataset archive yok.
* [ ] Benchmark dosyaları yalnız küçük CSV/JSON/Markdown.
