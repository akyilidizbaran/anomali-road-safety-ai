# Governance Alanı

Bu klasör etik, güvenlik, KVKK ve açık soru takibi için ayrılmıştır.

## Alt Klasörler

* `kvkk`: Kişisel veri ve KVKK notları.
* `security`: Erişim, storage ve API güvenliği.
* `open_questions`: Karar bekleyen sorular.

## Kural

Plaka, yüz, araç içi görüntü ve sürücü/yolcu bilgileri kişisel veri riski taşıyabilir. Bu başlıklar rapor ve demo öncesi ayrıca değerlendirilmelidir.
