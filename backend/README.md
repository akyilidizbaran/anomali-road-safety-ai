# Backend Alanı

Bu klasör edge/backend geliştirme planı ve ileride kodları için ayrılmıştır.

## Alt Klasörler

* `api_contracts`: REST/WebSocket contractları.
* `inference_server`: FastAPI/AI inference server.
* `evidence_store`: Evidence dosya/metadata saklama sistemi.

## İlk MVP

* `WS /stream`.
* `GET /events/recent`.
* `GET /events/{event_id}`.
* `GET /system/status`.
