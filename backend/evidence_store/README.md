# Evidence Store

Evidence storage implementasyonu ileride bu klasör altında tutulacaktır.

## Saklanacak Mantıksal Parçalar

* Event JSON.
* Overlay metadata.
* Image/crop URI.
* Model version.
* Decision reason.
* QoD status.

## Public Repo Rule

Gerçek evidence görselleri ve videoları public repoya commit edilmez.
